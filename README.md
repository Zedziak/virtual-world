# Symulator świata wirtualnego
Projekt przedstawia turową symulację świata, w którym różne gatunki organizmów (zwierzęta i rośliny) wchodzą ze sobą w interakcje na dwuwymiarowej planszy. Implementacja została wykonana w trzech językach: C++, Java oraz Python, aby zademonstrować różnice w podejściu do programowania obiektowego.
